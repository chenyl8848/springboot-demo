# SpringBoot 集成各种开发场景

## 项目介绍

这是一个汇集 SpringBoot 集成各种开发场景的项目，包含集成 Swagger、统一返回结果、全局异常处理、参数校验、多源数据源、各种中间件等等...
除了代码实现，在每一个模块的 README 文件中并配套相关的文章说明。
